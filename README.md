# EasyBarbearia

Plataforma completa para gestão de barbearias modernas.

## Funcionalidades

- Site profissional para barbearias
- Landing page para captação de clientes
- Sistema de gestão completo (agendamento, financeiro, estoque)
- Suporte a múltiplas unidades (plano Enterprise)
- Integração de pagamentos transparente via Mercado Pago

## Tecnologias

- Next.js
- React
- Tailwind CSS
- Lucide React
- Mercado Pago SDK

## Desenvolvimento

Para iniciar o servidor de desenvolvimento:

```bash
npm install
npm run dev
```
